# SteamGrid Overlay

Download game icons from SteamGridDB and apply console-specific overlays.

## Features

- Search and download game icons from SteamGridDB
- Apply console-specific overlays (PS3, Switch, NDS, etc.)
- Modern glassmorphism UI
- Optimized for landscape mode (perfect for gaming handhelds)
- Built for Ayn Odin Thor and similar devices

## Desktop Usage
```bash
python3 main.py
```

## Android APK

APK builds automatically via GitHub Actions. Download from the Releases page.

## Requirements

- Python 3.11+
- Kivy
- Pillow
- requests

See `buildozer_ayn_thor.spec` for full Android build configuration.
